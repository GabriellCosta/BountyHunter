Download
--------

add jitpack to your main build.gradle repositories and BountyHunter to your dependencies
with classpath

```gradle
buildscript {
    repositories {
        maven { url 'https://jitpack.io' }
    }

    dependencies {
        classpath 'com.github.GabriellCosta:BountyHunter:$version'
    }
}
```
